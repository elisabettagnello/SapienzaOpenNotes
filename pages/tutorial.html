🚀 Tutorial per collaborare

Una guida semplice per contribuire a Sapienza Open Notes

1️⃣ Crea un account GitHub

Se non hai già un account, registrati gratuitamente su github.com.

2️⃣ Fai un fork del progetto

Vai alla repository ufficiale: 👉 Sapienza Open Notes su GitHub .

Premi sul pulsante Fork in alto a destra: creerai una copia del progetto nel tuo profilo.

3️⃣ Scarica il progetto in locale

Hai due possibilità:

💻 Metodo 1: riga di comando (Git)

git clone https://github.com/TUO_USERNAME/SapienzaOpenNotes.git
cd SapienzaOpenNotes

🖱️ Metodo 2: strumenti grafici

Puoi usare un software come GitHub Desktop o SourceTree.

Ti basta inserire l’URL della tua copia del repository https://github.com/TUO_USERNAME/SapienzaOpenNotes.git e scegliere la cartella dove salvarlo.

4️⃣ Crea un branch per le tue modifiche

Anche qui puoi scegliere:

  • Git (riga di comando): 
    git checkout -b nome-del-tuo-branch
  • Strumenti grafici: usa l’opzione “New Branch” o “Checkout branch”.

5️⃣ Aggiungi o modifica i file

Segui la struttura del repository e le linee guida. Puoi aggiungere nuove lezioni, immagini o correggere errori.

6️⃣ Salva e invia le modifiche

💻 Con Git

git add .
git commit -m "Descrizione breve delle modifiche"
git push origin nome-del-tuo-branch

🖱️ Con strumenti grafici

Troverai pulsanti come Commit e Push. Inserisci un messaggio chiaro per descrivere le modifiche e inviale al tuo fork.

7️⃣ Apri una Pull Request

Vai sulla tua copia della repository su GitHub e premi Compare & Pull Request. Spiega cosa hai fatto e invia la richiesta: gli altri potranno revisionarla e unirla al progetto principale.

🎉 Fatto!

Hai contribuito a Sapienza Open Notes e hai aiutato la comunità a crescere 💡
